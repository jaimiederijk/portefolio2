Concerto concept webshop

Concerto is een grote muziek winkel in Amsterdam. De HVA heeft ons projectteam gevraagd om een nieuw webshop concept neer te zetten die Concerto naar het web brengt. Bij de klanten van concerto bleek de verpakking van de muziek erg belangrijk. Het zijn immers verzamelaars die hun muziek nog in een fysieke winkel halen. Om deze klanten ook buiten Amsterdam te kunnen bedienen hebben wij een webshop ontworpen die de verpakking centraal stelt door de producten als 3d modellen te presenteren. Mijn taak in dit project was fronend developer en ik heb dit concept uit mogen werken.

Mijn ontmantel installatie

De mijnontmantel installatie heb ik gemaakt voor een project waarbij ik een interactieve installatie moest maken voor een goed doel. Het goede doel werdt het Actiefonds Mijnen Ruimen. De installatie is een model van een m16 antipersoneel mijn gekoppelt aan een laptop via een Makey Makey. Het doel van de installatie is om de mijn te ontmantelen zonder hem af te laten gaan. Dit heb ik geprogrameerd met javascript.
